









# Project X - AI Module
Данный модуль предназначен для поиска людей по изображению с камеры
## Запуск
Через ```start.sh```
## Настройка
Настройки приложения расположены в файле ```.env```:
```
TOKEN=<TOKEN>
DATABASE=projectx
DATABASE_USER=projectx
DATABASE_PASS=projectx_pass
DATABASE_IP=127.0.0.1
DATABASE_PORT=3306
```
